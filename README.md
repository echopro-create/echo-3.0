# ECHO 4.0 (Next.js 15 + TypeScript + Tailwind)

Русский интерфейс. Только светлая тема. Минималистичный дизайн. Мобильная версия полноценна, но не первична.

## Что внутри
- App Router (Next.js 15), строгий TypeScript
- Tailwind (светлая тема, аккуратная типографика)
- Готовые страницы: главная, онбординг, логин (OTP), послания (список/новое), получатели, настройки, журнал, публичный просмотр
- Базовые API-роуты и клиент Supabase (SSR + client)
- Пример схемы БД и RLS (`/sql/schema.sql`)
- Чистые заглушки для рекордера (аудио/видео) и загрузчика файлов в Supabase Storage
- Кастомные страницы 404/500, строгие заголовки безопасности

## Быстрый старт
1) Создайте проект Supabase (EU). Включите Email OTP.
2) Заполните `.env` по образцу `.env.example`.
3) Импортируйте `/sql/schema.sql` в Supabase, включите RLS.
4) Залейте репозиторий на GitHub и подключите к Vercel. Укажите ENV там же.
5) Готово: деплой через пуш в `main`.
