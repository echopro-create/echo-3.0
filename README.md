# ECHO 3.0 (Etalon)

Готовый каркас проекта по «эталонному» ТЗ.

## Разработка
```bash
npm i
npm run dev
```

## Билд
```bash
npm run build
```

## Что внутри
- Страницы: `/`, `/login`, `/messages`, `/messages/new`, `/recipients`, `/settings`, `/about`, `/how`, `/security`, `/terms`, `/privacy`
- Плавная анимация текста на главной (последовательность: заголовок, подзаголовок, CTA)
- Заглушки API и клиентов Supabase
- Статус-панель и таймлайн (заглушки)
- Статические секции ниже первого экрана
```
